Go Boilerplate

🔵 Go 1.21+ 🟣 Echo v4 🐘 PostgreSQL 🔴 Redis 🟢 New Relic 🔒 Clerk Auth 📦 Turborepo

A production-ready, full-stack monorepo boilerplate built with Go and TypeScript. It comes pre-configured with authentication, observability, background jobs, transactional emails, database migrations, and a comprehensive testing framework — so you can focus on building your product.

What's Included

🌐 HTTP Server

Echo v4 with configurable timeouts, CORS, security headers, and graceful shutdown.

🔐 Authentication

Clerk SDK integration with JWT validation, session claims, and role-based access.

🗄️ Database

PostgreSQL with pgx v5 connection pooling, embedded tern migrations, and query tracing.

⏳ Job Queue

Asynq (Redis-backed) with priority queues, retries, and configurable concurrency.

📧 Emails

Resend API with Go HTML templates and asynchronous sending via the job queue.

📊 Observability

New Relic APM with distributed tracing across HTTP, DB, Redis, and log forwarding.

📡 Generic Handlers

Go generics-based handler system with automatic validation, logging, and tracing.

🛡️ Error Handling

Structured errors with automatic SQL-to-HTTP mapping and user-friendly messages.

🧪 Testing

Testcontainers for real PostgreSQL integration tests with per-test isolation.

📦 TypeScript

Zod schemas, ts-rest OpenAPI generation, and React Email templates in a monorepo.

🔗 Middleware

9-layer pipeline: rate limiting, CORS, request IDs, auth, tracing, and logging.

⚙️ Configuration

Type-safe config from env vars with koanf, validation, and sensible defaults.

Prerequisites

Before getting started, make sure you have the following tools installed:

ToolVersionInstall CommandPurpose
Go≥ 1.21 brew install go or go.dev/dl Backend language runtime and compiler
Node.js≥ 22 brew install node or nodejs.org Required for TypeScript packages (Zod, OpenAPI, emails)
Bun≥ 1.3 curl -fsSL https://bun.sh/install | bash Fast JS package manager used by the monorepo
PostgreSQL≥ 15 brew install postgresql Primary database for all application data
Redis≥ 7 brew install redis Backend for background job queue (Asynq)
Task≥ 3 brew install go-task Task runner for build, test, lint, and run commands
DockerLatest docker.com Required only for running integration tests (Testcontainers)
golangci-lintLatest brew install golangci-lint Go linter aggregator for code quality checks

See Getting Started for detailed installation and setup instructions.

Quick Start

# Clone & install
git clone https://github.com/yaswanth-byk/go-boilerplate.git
cd go-boilerplate
bun install
cd apps/backend && go mod download

# Configure
cp .env.sample .env
# Edit .env with your database, Clerk, and Resend credentials

# Run
task run
